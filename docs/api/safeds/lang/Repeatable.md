# `#!sds annotation` Repeatable {#safeds.lang.Repeatable data-toc-label='Repeatable'}

The annotation can be called multiple times for the same declaration.

**Targets:**

- `Annotation`
- `Attribute`
- `Class`
- `Enum`
- `EnumVariant`
- `Function`
- `Module`
- `Parameter`
- `Pipeline`
- `Result`
- `Segment`
- `TypeParameter`
